## Overview

Aviation Technical Content Manager (ATCM) is an enterprise solution for managing illustrated parts catalogs (IPCs) and technical documentation for aviation and defense operators. I contributed to building content management workflows, search capabilities, and integration with aircraft maintenance systems.

## Technical Contributions

### Content Management System
- Developed catalog management workflows for technical publications and illustrated parts catalogs
- Built version control system for tracking changes to technical documentation across aircraft models
- Implemented approval workflows for catalog updates, ensuring compliance with aviation regulations

### Search & Retrieval
- Designed and implemented search engine for thousands of technical illustrations and part specifications
- Built indexing system for part numbers, descriptions, and cross-references across multiple aircraft types
- Developed filtering and navigation UI for maintenance technicians to quickly locate required parts

### Quality Assurance & Performance
- Served as QA Champion, managing a team of 7 developers and establishing testing frameworks
- Became go-to person for resolving critical customer-focused bugs, ensuring timely delivery
- Resolved critical performance issues and optimized database queries (Oracle PL/SQL), reducing lookup time by 60%

### Integration & Collaboration
- Integrated ATCM with IFS Fleet Planner and maintenance tracking systems
- Built AWS-based document storage and retrieval layer for technical illustrations
- Collaborated closely with global teams across Canada, Sweden, and UK for feature development

## Architecture

Built on IFS Aurena framework with Oracle database backend.

## Impact

- Enabled rapid parts lookup for maintenance operations across multiple airline clients
- Supported global collaboration with seamless integration across IFS teams in Canada, Sweden, and UK
- Improved catalog accuracy through structured version control and approval processes

## Technologies

**Backend:** .NET C#, Oracle PL/SQL  
**Platform:** IFS Aurena Framework  
**Database:** Oracle Database
