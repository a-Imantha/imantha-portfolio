# Keycloak

Enterprise-grade open-source identity and access management. Provides SSO, identity brokering, and user federation.

## Key Features

- **Single Sign-On**: Centralized authentication
- **OAuth 2.0 & OpenID**: Standard protocols
- **User Federation**: LDAP and Active Directory
- **Identity Brokering**: Social login integration
- **Fine-grained Authorization**: Role-based access control

## My Experience

Implemented authentication and authorization systems with Keycloak, managing user access across microservices and applications.
