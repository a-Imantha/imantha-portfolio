# RunPod

Cloud computing platform specialized in GPU infrastructure for AI/ML workloads. Provides instant access to high-performance GPUs at competitive prices.

## Key Features

- **GPU Selection**: Wide range of GPU options (A100, H100, RTX series)
- **Instant Deployment**: Quick spin-up of GPU instances
- **Flexible Pricing**: Per-hour or long-term rentals
- **Pre-configured Templates**: Ready-to-use ML environments
- **Serverless GPU**: On-demand GPU execution

## My Experience

Used RunPod for renting GPUs during the LLM fine-tuning process for Chandra X-ray Observatory data analysis, enabling cost-effective access to powerful compute resources.
